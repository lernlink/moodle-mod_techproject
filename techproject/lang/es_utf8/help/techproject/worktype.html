Cet indicateur permet de qualifier le type de tâche à fournir. Connaître le type de tâche peut fournir des indications pour la répartition des tâches entre les participants au projet.

Les valeurs données en standard concernent un projet de développement logiciel classique, mais toute autre échelle propre à un projet particulier peut être construite par l'enseignant.

Valeurs par défaut :

  • PRE Préétude : Tâches de recherche d'information et de collecte d'information
  • ANA Analyse : Analyse de solution, modélisation
  • SPK Spécifications : Ecritures de specification
  • COD Codage : Ecriture de code
  • REV Revues : Revue de code, harmonisation, nettoyage
  • PAK Packaging : Packaging d'application, installers, distributions
  • DOC Documentation : Ecriture de documentations
