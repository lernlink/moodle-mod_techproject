Cet indicateur détermine l'état d'un livrable. Il peut prendre un certain nombre de valeurs qui désignent un certain moment de son cycle de vie :

Les valeurs par défaut sont :

  • CREATED : Le livrable est créé comme enveloppe/packaging mais est vide
  • BETA : Le livrable n'est pas distribuable comme produit fini
  • FIX : Le livrable contient des défaut en cours de correction.
  • OBSOLETE : Le livrable ne contient plus une version utilisable du produit.
  • TEST : Les contenus du livrable sont en cours de test.
  • PACKAGING : Le livrable est en cours de finition. Les opérations en cours concernent des éléments du packaging final tels que documentation, jeux d'essai, démonstrateurs, etc.
  • REVIEW : Les contenus du livrable sont en cours de relecture/revue de code/vérification.
  • STABLE : Le produit est en cours de distribution sur la base de ce contenu.
