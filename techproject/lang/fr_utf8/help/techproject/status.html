Cet indicateur détermine l'état d'une tâche. Il peut prendre un certain nombre de valeurs qui désignent un certain moment de son cycle de vie :

Les valeurs par défaut sont :

  • PLANNED : La tâche est planifiée
  • STARTED : La tâche est démarrée, elle a été prise en charge par le contributeur qui en prend la responsabilité.
  • BLOCKED : La tâche est bloquée par une contrainte extérieure. Débloquer cette tâche suppose éliminer la raison externe pour laquelle cette tâche ne peut être achevée.
  • DELAYED : La tâche est momentanément différée pour des raisons internes d'organisation du travail.
  • COMPLETE : La tâche est achevée. Le travail est considéré comme terminé, quelque soit l'état de finition de l'objectif.
  • ABANDONED : La tâche est abandonnée. Le travail est considéré n'ayant plus d'intérêt.
