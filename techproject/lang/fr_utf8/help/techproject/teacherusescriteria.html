Si cette option est activée, la notation du projet s'effectue à travers une grille de critères définie par l'enseignant.

Vous pouvez modifier cette grille par le panneau approprié dans les "outils de l'enseignant".

Les critères peuvent être modifiés même si des notes ont été données. Par contre, si un critère est supprimé à ce moment, les notes qui lui sont associées seront perdues.
